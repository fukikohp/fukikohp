📸 Instagram フィード自動反映 — セットアップ手順書

この手順書では、Instagram Graph API のアクセストークンを取得して、 ホームページに Instagram の最新投稿を自動表示させる方法を説明します。

⚠️ 事前準備（必須）
Instagram アカウントを 「プロアカウント（ビジネス or クリエイター）」 に変更する必要があります。
個人アカウントのままでは 2024年12月以降の API には対応していません。

STEP 1 Instagramをプロアカウントに切り替える

  1. Instagram アプリを開き、プロフィールページへ
  2. 右上の「≡（ハンバーガーメニュー）」→「設定とアクティビティ」
  3. 「アカウント」→「アカウントの種類とツール」
  4. 「プロアカウントに切り替える」→「クリエイター」または「ビジネス」を選択
  5. カテゴリを選んで完了
📌 フォロワーや投稿はそのまま引き継がれます。見た目も変わりません。

STEP 2 Meta for Developers でアプリを作成する

  1. https://developers.facebook.com/ にアクセス
  2. 右上「マイアプリ」→「アプリを作成」
  3. 「その他」を選択 → 「次へ」
  4. アプリ名（例：fukiko-website）を入力 → 「アプリを作成」
📌 Facebook アカウントでログインが必要です。持っていない場合は先に作成してください。

STEP 3 Instagram API を有効にする

  1. 作成したアプリのダッシュボードが開く
  2. 左メニュー「製品を追加」の中から 「Instagram」 を探して「設定」をクリック
  3. 「Instagram ビジネスログインを使用した API 設定」を選択

STEP 4 アクセストークンを生成する

  1. 左メニュー「Instagram」→「API setup with Instagram business login（APIの設定）」
  2. 「Instagram アカウントを追加する」をクリック
  3. Instagram にログインして連携を許可
  4. アカウントが表示されたら、横の 「トークンを生成」 ボタンをクリック
  5. 表示された 長いトークン文字列 をコピーしてください
⚠️ このトークンは 60日間有効 です。期限前に更新が必要です（STEP 6 参照）。

STEP 5 index.html にトークンを貼り付ける

index.html をテキストエディタで開き、以下の部分を探してトークンを貼り付けます。

// ★ ここにアクセストークンを貼り付けてください ★
const INSTAGRAM_ACCESS_TOKEN = 'ここにコピーしたトークンを貼り付ける';

例：

const INSTAGRAM_ACCESS_TOKEN = 'EAABsbCS1iHgBABCDEFG12345...';

保存してブラウザでページを再読み込みすると、最新9件の Instagram 投稿が表示されます。

STEP 6 トークンを更新する（60日ごと）定期作業

アクセストークンは 60日 で失効します。失効すると投稿が表示されなくなります。
以下の URL をブラウザで開くと、新しいトークンが取得できます。

https://graph.instagram.com/refresh_access_token?grant_type=ig_refresh_token&access_token=【現在のトークン】

返ってきた JSON の access_token の値を、STEP 5 と同じように貼り直してください。

💡 おすすめ： スマホのカレンダーに「Instagramトークン更新」を55日後に繰り返しリマインダーとして設定しておくと安心です。

✅ セットアップ完了チェックリスト

  • ☐ Instagram をプロアカウント（クリエイター / ビジネス）に変更した
  • ☐ Meta for Developers でアプリを作成した
  • ☐ Instagram API を有効にした
  • ☐ アクセストークンを生成してコピーした
  • ☐ index.html の INSTAGRAM_ACCESS_TOKEN にトークンを貼り付けた
  • ☐ ブラウザで投稿が表示されることを確認した
  • ☐ カレンダーに55日後のトークン更新リマインダーを設定した

❓ よくある質問

Q. 投稿が表示されない場合は？

  • トークンが正しく貼り付けられているか確認する
  • Instagram アカウントがプロアカウントになっているか確認する
  • トークンの有効期限（60日）が切れていないか確認する
  • Instagram アカウントが「公開」設定になっているか確認する

Q. 個人アカウントのままではダメ？

2024年12月に Basic Display API が廃止されたため、現在はプロアカウント（ビジネス / クリエイター）のみ対応しています。

Q. 何枚表示されますか？

最新 9件 の投稿が表示されます。index.html の const POST_COUNT = 9; の数字を変えると変更できます（最大12）。

Q. 動画（リール）も表示されますか？

はい。動画はサムネイル画像で表示され、クリックすると Instagram のページに飛びます。
